Collatz Conjeture (CAC 2023)

Collatz Conjecture Calculator

Description

My Collatz Conjecture Calculator was my submission to the 2023 Congressional App Challenge. It takes a numerical input from the user and cycles the number through the algorithm on which the Collatz Conjecture is based. It then returns the number of cycles before it reaches "the loop" as well as the highest value reached.
